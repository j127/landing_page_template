Landing Page

This landing page template works with SCSS and TypeScript. If you prefer JavaScript, change the .ts file to .js and its reference in the index.html file. See the package.json file for a list of scripts. (E.g., build the site with yarn build, deploy the site with yarn deploy.)

Everything in the Github repo is public domain, including the code and images.

Replace the water.css import in index.html with any CSS framework (Bootstrap, Bulma, etc.). The water.css file just makes the default page look nicer without needing any CSS classes. Read more about water.css here.
