Landing Page

Works with SCSS and TypeScript. If you prefer JavaScript, change the .ts file to .js and its reference in the index.html file. See the package.json file for a list of scripts. (E.g., build the site with yarn build, deploy the site with yarn deploy)

Replace the water.css import in index.html with any CSS framework. The water.css file just makes the default page look nicer without needing any CSS classes.
